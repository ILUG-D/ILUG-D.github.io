SUMMER PROJECT 2019 - ILUG-D

Hey folks, We are hosting a bunch of community projects for the summer. If you’re in for some real challenge this holiday season and commit some time to solve a problem with open source solution, the time has come !! We bet, this will be an amazing learning experience which will also leave its mark on your open source profile ;) We have planned to make a meetup automation api and make webapp and android apps for the same. These are the modules that the project has been broken down into:

  • Birdman NotASPAMbox - Automated mail and social media publishing command line interface

  • Splatter - Poster generator in Golang

  • Välkommen - automated attendance and feedback retrieval system

  • Scribe - RPi based video recorder>

You may go ahead, and start making contributions to these modules. Further details about the project and implementation are presented in the README.md. For people starting out in open source, this is going to be the best starting point. These are the roles which you can take up :

  • Developers

  • Testers

  • Documentation maintainers

  • Mentors
